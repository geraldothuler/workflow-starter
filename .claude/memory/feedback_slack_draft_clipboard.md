Ao gerar qualquer mensagem Slack: apresentar o texto no chat e copiar para o clipboard via `pbcopy`. Não enviar diretamente — o usuário prefere postar por conta própria.

**Why:** prefere controle sobre o que é enviado, especialmente em threads privadas.

**How to apply:** sempre que o usuário pedir "versão pra Slack", "manda no Slack", "escreve pra Slack" — apresentar + `pbcopy`. Perguntar canal/thread só se o usuário quiser que eu envie.
